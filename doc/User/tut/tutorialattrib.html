Tutorial Attribute

Intro Scale Square Smooth

Intro

Here, you can perform the same operations on seismic data as in the Tut Seismic Tools, but the output in this case is an attribute instead of another seismic cube. From the user point of view the biggest advantage is that you can apply this attribute on another attribute or alternatively, you can use this output attribute as input for another attribute. From the OpendTect menu, select


Processing --> Attributes

Or click on the Edit Attributes icon in the toolbar to open the Attribute Editor dialog. From the drop-down list, select the Tutorial attribute. In the Input Data selection field, select an existing seismic cube or a pre-defined attribute. You can choose from three available operations in the Action field and accordingly specify the corresponding input parameters:

Scale

It performs a linear scaling of the form a + b * x, where parameters a and b are taken from the inputs Shift and Factor respectively.

Square

As the name suggests, it takes the square of data values.

Smooth

Here, we have got two different types of smoothing: Horizontal and Vertical.

The Vertical smoothing takes samples above and below the data sample along the same trace and is similar to that in Tut Seismic Tools and produces the same results. You can choose a Low or High filter strength to specify whether 3 or 5 samples are taken for calculating the mean.

The Horizontal smoothing involves computation using samples from the neighbouring traces. If you want the selection of samples to be guided by a steering cube, you can select one in the Steering Data field by choosing Full or Central option in the Steering field. A rectangle centred around the data sample is taken with the size of the rectangle specified by the Stepout. The output value is the arithmatic mean of all the values within this rectangle.
