Tut Seismic Tools

Intro Scale Square Smooth

Intro

Here, you can modify a seismic cube by accessing the seismic traces and manipulating their values directly. From the OpendTect menu, select


Utilities --> Tut Tools --> Seismic (Direct)

In the Input Cube selection field, select an existing seismic cube. For the Output Cube, you can either select an existing cube or enter the name of a new seismic cube. You can choose from three available operations in the Action field and accordingly specify the corresponding input parameters:

Scale

It performs a linear scaling of the form a + b * x, where parameters a and b are taken from the inputs Shift and Factor respectively.

Square

As the name suggests, it takes the square of data values.

Smooth

This involves a vertical smoothing of traces. You can choose a Low or High filter strength to specify whether 3 or 5 samples are taken for calculating the mean.
