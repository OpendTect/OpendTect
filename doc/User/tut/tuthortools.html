Tut Horizon Tools

Intro Smoothing Thickness Calculation

Intro

Here, you can perform some simple operations on horizon surfaces like Smoothing and Thickness calculation. From the OpendTect menu, select


Utilities-->Tut Tools-->Horizon

You can choose from two available operations in the Select Task field: Smooth a Horizon and Find Thickness between two Horizons.

Smoothing

It takes and Input Horizon as input and performs simple smoothing on it, The Output Horizon can either be an existing horizon or a new one. For smoothing, a square centred around each point is taken and the z value of the point is replaced by the average of z values of all the points lying within the square. The size of the square depends on the Filter Strength which can be chosen as Low or High.

Thickness Calculation

This utility can be used where the thickness between two surfaces is needed, for example in Isopach map computation. It takes two horizons as input. The terms Top and Bottom are only symbolic and have no bearing on the result, as the thickness is taken as the absolute difference between the z values of corresponding points on the two surfaces, and hence is always positive. The ouput is stored as an attribute to one of the input horizons, specified by you. You can also give a name for this attribute which is named Thickness by default.
